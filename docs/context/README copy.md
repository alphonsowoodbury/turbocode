# Context

AI-Enhanced Personal Knowledge System

## Core Philosophy
- Zero-friction entry & retrieval – Instant capture, blazing-fast search
- Privacy-first – End-to-end encrypted, local-first, syncs only when necessary
- Utility over engagement – No dopamine loops, only high-value interactions
- Apple-grade design & UX – Seamless, elegant, invisible complexity
